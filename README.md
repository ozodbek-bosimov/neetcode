# NeetCode Solutions

My personal solutions to NeetCode problems.

## Structure

```
<topic>/
  <problem-slug>/
    submission-0.<ext>
    submission-1.<ext>
```

## Example

```
data-structures-and-algorithms/
  two-integer-sum/
    submission-0.py
  binary-search/
    submission-0.go
```

## Notes

* Each folder represents a topic or category.
* Problems are organized by slug (LeetCode-style naming).
* Multiple submissions may exist for practice and optimization.
